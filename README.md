# MongoDB Assignment

## Overview
- Work with MongoDB collections
- Insert nested JSON data
- Query nested fields in MongoDB
- Integrate MongoDB with PySpark
- Perform transformations using Spark DataFrame
- Write processed data back to MongoDB

## Technologies Used
- MongoDB Atlas
- Mongo Shell (mongosh)
- PySpark
- MongoDB Spark Connector
- Python
